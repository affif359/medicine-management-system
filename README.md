# medicine-management-system
---MEDICINE MANAGEMENT SYSTEM---

The Medicine Management System is a console-based application designed to modernize how medical shops handle their inventory.

It replaces error-prone manual ledgers with a fast, digital way to track medicine stock, ensuring that vital information is always just a few keystrokes away.

---FEATURES---

this will help the user to choose 4 options.

1.adding any medicine to the inventory.

2.deleting any unnecessary medicine from the inventory.

3.searching any medicine whether it is present or not in the list.

4.displaying of the current inventory.

---Technologies and Tools Used---

This project adheres to a lean architecture to prioritize reliability and portability.

| Component | Technology/Format | Description |
| :--- | :--- | :--- |
| Programming Language | [Specify: Python 3.8+] | Chosen for strong file handling capabilities and stability in a CLI environment. |
| Interface | Command Line Interface (CLI) | Used for simplicity and rapid execution speed. |
| Data Storage | File Handling (Text) | All inventory records are saved and loaded from a local file (`inventory_records.csv`) to ensure data persistence. |
